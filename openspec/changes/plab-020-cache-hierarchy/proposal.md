# PLAB-020: Cache Hierarchy

## Summary
Build an interactive laboratory explaining CPU cache hierarchy (L1/L2/L3/RAM), locality, cache lines, latency vs bandwidth, prefetching and cache-aware software design.

## Scope
- L1/L2/L3/Main Memory
- Cache lines
- Spatial & temporal locality
- Sequential vs random access
- Hardware prefetching
- Java (JMH)
- Rust (Criterion)
- Interactive visualization
- Benchmark methodology
