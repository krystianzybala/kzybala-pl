# Requirements

## Requirement: Explain CAS

The lab MUST explain compare-and-set as a conditional atomic update.

## Requirement: Contention model

The lab MUST show how retry loops amplify coherence traffic and latency under contention.

## Requirement: Backoff

The lab MUST compare no backoff, fixed backoff, exponential backoff, and jitter.

## Requirement: Alternatives

The lab MUST compare CAS contention with a single-writer ownership model.

## Requirement: Java and Rust

The lab MUST include buildable Java and Rust examples and comparable benchmark methodology.

## Requirement: Caveats

The lab MUST explain fairness, starvation risk, ABA at a conceptual level, and workload sensitivity.
